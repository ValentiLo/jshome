  3. для чего нужен оператор switch:
    Оператор switch javascript служит для проверки переменной на множество значений
  4. что делается в case
    в каждом case проверяется одно из значений, в случае подходящего значения выполняется тот или иной блок операторов, соответствующий данному case.
  5. Для чего нужен break
    Оператор break обязателен после каждого рассмотренного значения переменной (после каждого case); если его не использовать, то будут выведены все, расположенные ниже операторы
  6. Синтаксис оператора switch. После каких слов ставится двоеточие
    switch(a)
    {
    case 0: // if (a === 0)
    case 1: // if (a === 0)
    alert("Ноль или один"); // тогда выводим ...
    break;
    case 2: // if (a === 2)
    alert("Два"); // тогда выводим ...
    break;
    default: // else
    alert("Много"); // иначе выводим ...
    }
