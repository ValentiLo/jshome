94. Пример: Используя функцию, необходимо вычислять возведение числа в степень. Выполнить задание, используя цикл for
Показать решение:
	let chislo,stepen;
function degree(chislo,stepen) { 
        for(let result = 1; stepen > 0; stepen--) {
                result *= chislo; 
        } 
return  result;
} 
document.write(degree(2,4)); // выводит 16
