125. Метод javascript push(значение) — добавляет к массиву указанное значение в качестве последнего элемента и возвращает новую длину массива
имя_массива.push(значение)
Пример использования метода push:
let x = ['a', 'b', 'c', 'd'];
document.write(x.push('e'));
//выведет на экран обозревателя число 5
