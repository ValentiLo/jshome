Метод javaScript unshift(значение) — добавляет к массиву указанное значение в качестве первого элемента. Возвращает новую длину массива
имя_массива.unshift(значение)
124. Пример использования метода unshift:
let x = ["a", "b", "c", "d"];
document.write(x.unshift("e"));//выведет на экран обозревателя число 5
