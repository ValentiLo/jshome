Пример 44: Запрашивать у пользователя ввести цвет. Выводить перевод на английский язык введенного цвета. Для цвета "синий" и "голубой" выдавать одно и то же значение.

Решение:

Создайте веб-страницу с html-скелетом и тегом script.
Инициализируйте переменную color значением, введенным пользователем в модальное окно:
let color = prompt("Какой цвет?");
Проверьте значение переменной с помощью конструкции switсh, выводя на каждое значение - соответствующий перевод:
switch (color)
{ case "красный" :
alert("red");
break;
case "зеленый":
alert("green");
break;
// ...
Если переменная color имеет значение "красный", то вывести в модальное окно перевод - "red" и выйти из конструкции (break;). Если переменная color имеет значение "зеленый", то вывести в модальное окно перевод - "green" и выйти из конструкции (break;). Для цветов "синий" и "голубой" выполните группировку:
// ...
case "синий":
case "голубой":
alert("blue");
break;
// ...
Если переменная color имеет значение "синий" или переменная color имеет значение "голубой", то вывести в модальное окно перевод - "blue" и выйти из конструкции (break;). Организуйте вывод для тех цветов, которые не предусмотрены программой: // ...
default:
alert("y нас нет сведений по данному цвету")
} // конец switch
Протестируйте сценарий в браузере.

  1. Как сгруппировать несколько вариантов? Для синего и голубого один результат (перевод)
    ...
    case 0:
    case 1:
    alert("Ноль или один");
    break;
    ...
    При a = 0 и a = 1 выполняется один и тот же оператор:
    alert("Ноль или один");
  2. Что делает let color = prompt("Какой цвет?");
    Инициализируйте переменную color значением, введенным пользователем в модальное окно:
  3. case "красный" :
    alert("red");
    Если переменная color имеет значение "красный", то вывести в модальное окно перевод - "red" и выйти из конструкции (break;).
  4. Что происходит
    case "синий":
    case "голубой":
    alert("blue");
    Если переменная color имеет значение "синий" или переменная color имеет значение "голубой", то вывести в модальное окно перевод - "blue" и выйти из конструкции (break;).
