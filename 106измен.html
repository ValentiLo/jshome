Обращение или доступ к элементам массива в javaScript происходит так:
        106. let mas=new Array(1,25,'Привет');
        mas[0]='Пока';
        mas[1]=35;
        При инициализации массива элемент mas[0] был равен 1. Затем мы изменили его значение на ‘Пока’. Значение элемента массива mas[1] было изменено с 25 на 35.document.write(mas)
        Вывод элементов массива
        При выводе значений массива в документ с помощью метода write, его элементы выводятся через запятую:
        let mas=new Array(1,25,'Привет');
        document.write(mas)
        Результат:           1,25,Привет
